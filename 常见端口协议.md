# 常见端口+协议完整对照表
## 一、基础概念
1. **端口**：0–65535，分三类
   - 知名端口：0–1023（系统/标准服务）
   - 注册端口：1024–49151（软件自定义）
   - 动态私有端口：49152–65535（临时连接）
2. **传输层协议**
   - TCP：可靠连接、三次握手（网页、文件传输）
   - UDP：无连接、速度快（视频、DNS、游戏）

## 二、知名常用端口（必记）
### HTTP/网页类
| 端口 | 协议 | 服务说明 |
|------|------|----------|
| 80 | TCP | HTTP 明文网页 |
| 443 | TCP | HTTPS 加密网页（SSL/TLS） |
| 8080 | TCP | HTTP 代理/备用网页端口 |
| 8443 | TCP | HTTPS 备用加密端口 |

### 文件传输
| 端口 | 协议 | 服务说明 |
|------|------|----------|
| 21 | TCP | FTP 控制端口 |
| 20 | TCP | FTP 数据传输 |
| 22 | TCP | SFTP/SSH 文件传输（加密） |
| 69 | UDP | TFTP 简易文件传输（无认证） |

### 远程管理
| 端口 | 协议 | 服务说明 |
|------|------|----------|
| 22 | TCP | SSH 远程Linux登录 |
| 23 | TCP | Telnet 明文远程登录（不安全） |
| 3389 | TCP | RDP Windows远程桌面 |
| 5900 | TCP | VNC 远程图形桌面 |

| 110 | TCP | POP3 接收邮件（明文） |

| 143 | TCP | IMAP 邮件同步 |

|------|------|----------|

## 三、UDP典型高速服务
